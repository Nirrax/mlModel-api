## Description

FastAPI service for Music Genre Classification project  
[Second repo for the project](https://github.com/Nirrax/Auth-api)

## Requirements

Python version - 3.10.11  
FFMPEG

## Installation

```bash
$ pip install -r requirements.txt
```

## Running the app

```bash
$ uvicorn main:app

$ uvicorn main:app --reload
```
